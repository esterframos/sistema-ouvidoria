# Sistema de Ouvidoria

Projeto acadêmico desenvolvido durante o curso de Análise e Desenvolvimento de Sistemas.

## Descrição
Sistema simples de ouvidoria em Python, organizado em módulos, com menu principal e funções para registro e consulta de manifestações.

## Tecnologias Utilizadas
- Python

## Estrutura do Projeto
- menu.py: arquivo principal do sistema
- meria.py: funções e regras do sistema

## Objetivo
Aplicar conceitos iniciais de programação, lógica, funções (`def`) e organização de código.
